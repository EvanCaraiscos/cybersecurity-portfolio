# A28 - Security Solution Implementation

## Solution
Password Strength Checker

## Repository
https://github.com/EvanCaraiscos/Password_strength_checker

## What it does
A Python-based tool that evaluates the strength of a 
password based on:
- Length
- Uppercase and lowercase letters
- Numbers
- Special characters

Classifies passwords as Weak, Moderate, or Strong and 
provides feedback on how to improve weak passwords.

## Why it's a security solution
Weak passwords are one of the leading causes of account 
compromise. This tool helps users understand whether their 
passwords meet basic security standards before using them, 
directly addressing one of the most common attack vectors —
credential based attacks.

## How to run
1. Clone the repo
2. Run: python password_strength_checker.py
