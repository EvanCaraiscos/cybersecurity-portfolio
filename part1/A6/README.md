# A6 - Cryptographic Implementation Used Offline

## Overview
Cryptographic implementations discovered in use 
in offline/local environments.

## Implementations Covered

| Implementation | Use Case |
|---|---|
| BitLocker | Full disk encryption on Windows |
| VeraCrypt | Encrypted containers and volumes |
| Password Hashing (bcrypt) | Storing passwords securely |
| PGP/GPG | Encrypting files and emails offline |
| WinZip/7-Zip AES encryption | Encrypting compressed files |
