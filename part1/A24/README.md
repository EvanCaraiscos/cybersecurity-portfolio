# A24 - Teach Family About Cybersecurity

## Overview
Taught my dad about password security and safe 
password storage, resulting in him adopting 
Bitwarden as a password manager.

## Topics Covered
- Why weak/reused passwords are dangerous
- How password managers work
- Setting up and using Bitwarden

## Outcome
Dad installed Bitwarden and moved away from 
reusing simple passwords across accounts.
