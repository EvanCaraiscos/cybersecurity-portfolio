# A17 - Ten Different Types of Locks in Use

## Locks Discovered

| Lock | Type | Domain |
|---|---|---|
| Deadbolt | Mechanical | Physical |
| Padlock | Mechanical | Physical |
| Combination Lock | Mechanical | Physical |
| Cam Lock | Mechanical | Physical |
| Mortise Lock | Mechanical | Physical |
| Bike Lock (D-Lock) | Mechanical | Physical |
| Account Lockout | Digital | Cyber |
| Cryptographic Key Lock | Digital | Cyber |
| Screen/Device Lock | Digital | Cyber |
| Network Access Control (NAC) | Digital | Cyber |

## Notes
A mix of physical and digital locks showing how the 
concept of "locking" applies across both domains.
