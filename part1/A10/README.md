# A10 - Privacy Technique Used Offline

## Technique: Faraday Bags

A Faraday bag is a physical privacy tool that blocks 
radio frequency signals, preventing wireless 
communication to and from devices placed inside it.

## Key Concepts Covered
- How Faraday cages work
- Use cases: key fobs, phones, passports, laptops
- Relay attacks on keyless entry vehicles
- Limitations
