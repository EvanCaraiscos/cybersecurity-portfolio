# A9 - Privacy Technique Used Online

## Technique: Virtual Private Network (VPN)

A VPN is a privacy technique that encrypts internet 
traffic and masks a user's real IP address by routing 
connections through a server in another location.

## Key Concepts Covered
- How VPN tunnelling works
- Encryption protocols used (OpenVPN, WireGuard, IKEv2)
- Use cases: privacy, geo-restriction bypass, public WiFi
- Limitations and trust considerations
