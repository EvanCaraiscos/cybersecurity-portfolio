# A15 - Five Recent Security Incidents

## Overview
Five significant cybersecurity incidents from 2025, covering
a range of attack types and sectors.

## Incidents

### 1. Qantas Data Breach — Australia (June 2025)
- **Type:** Third-party platform compromise, social engineering
- **Impact:** 5.7 million customers' personal data exposed
- **Source:** https://www.infosecurity-magazine.com/news/qantas-57m-data-breach/

### 2. Marks & Spencer Ransomware — UK (2025)
- **Type:** Ransomware via third-party social engineering
- **Impact:** Customer data exfiltrated, extortion demands issued
- **Source:** https://insights.integrity360.com/the-biggest-cyber-attacks-of-2025-and-what-they-mean-for-2026

### 3. Oracle E-Business Suite Zero-Day — Global (October 2025)
- **Type:** Zero-day exploit (CVE-2025-61882) by Clop ransomware group
- **Impact:** Multiple global organisations compromised including NHS trusts
- **Source:** https://www.infosecurity-magazine.com/news-features/top-10-cyberattacks-of-2025/

### 4. Coupang Data Breach — South Korea (December 2025)
- **Type:** Data breach, insider threat suspected
- **Impact:** 33.7 million customer accounts compromised
- **Source:** https://www.csis.org/programs/strategic-technologies-program/significant-cyber-incidents

### 5. French Interior Ministry Breach (December 2025)
- **Type:** Email server compromise
- **Impact:** Ministry email systems breached, security protocols tightened
- **Source:** https://www.csis.org/programs/strategic-technologies-program/significant-cyber-incidents
