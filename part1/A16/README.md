# A16 - Three Local Security Incidents

## Overview
Three significant cybersecurity incidents affecting 
Australian organisations and citizens.

## Incidents

### 1. MediSecure Data Breach (2024)
- **Type:** Ransomware
- **Impact:** 12.9 million Australians' health data exposed
- **Source:** https://www.abc.net.au/news/2024-05-16/medisecure-cyber-attack/103861304

### 2. Optus Data Breach (2022)
- **Type:** API vulnerability exploitation
- **Impact:** 9.8 million customer records exposed
- **Source:** https://www.abc.net.au/news/2022-09-22/optus-data-breach/101462282

### 3. Latitude Financial Breach (2023)
- **Type:** Credential theft / data exfiltration
- **Impact:** 14 million records stolen across AU and NZ
- **Source:** https://www.abc.net.au/news/2023-03-27/latitude-financial-hack-14-million-records/102149702
