# A7 - Cryptography Used in Modern Networks

## Overview
Cryptographic protocols discovered in use across 
modern network infrastructure.

## Protocols Covered

| Protocol | Use Case |
|---|---|
| TLS 1.3 | Encrypts web traffic (HTTPS) |
| WPA3 | Secures WiFi networks |
| SSH | Secure remote server access |
| IPSec | Encrypts VPN tunnels |
| DNS over HTTPS (DoH) | Encrypts DNS queries |
